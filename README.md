# Laventra Mobile — Setup Guide

Flutter mobile app for the Laventra car wash management platform.

## Requirements

- Flutter 3.41.x (stable channel)
- Dart SDK 3.11+
- Xcode 15+ (for iOS builds)
- Android Studio + Android SDK (for Android builds)
- A running `laventra_app` (Rails backend)

---

## 1. Clone & Install Dependencies

```bash
git clone <repo-url>
cd laventra_mobile
flutter pub get
```

---

## 2. iOS Setup (first time only)

```bash
cd ios
pod install
cd ..
```

Open `ios/Runner.xcworkspace` in Xcode, go to **Signing & Capabilities**, select your development team, then return to the terminal.

---

## 3. Firebase Setup

The app uses Firebase for push notifications. The Firebase config files are **gitignored** and must be obtained from the Firebase console.

**iOS** — download `GoogleService-Info.plist` and place it at:
```
ios/Runner/GoogleService-Info.plist
```

**Android** — download `google-services.json` and place it at:
```
android/app/google-services.json
```

> Without these files the app will build but push notifications will not work.

---

## 4. Run the App

**iOS:**
```bash
flutter run -d ios
```

**Android:**
```bash
flutter run -d android
```

**List available devices:**
```bash
flutter devices
```

---

## Files That Are Gitignored and Must Be Created/Obtained Manually

| File | How to Obtain |
|------|--------------|
| `ios/Runner/GoogleService-Info.plist` | Download from Firebase console for this project |
| `android/app/google-services.json` | Download from Firebase console for this project |
| `.dart_tool/` | Auto-generated by `flutter pub get` |
| `build/` | Auto-generated on build |
| `.flutter-plugins-dependencies` | Auto-generated by `flutter pub get` |

---

## Backend Dependency

The mobile app communicates with the Rails backend (`laventra_app`). For local development, update the base URL in the app's API config to your machine's local IP — devices and emulators cannot reach `localhost`.

Find your local IP:
```bash
ipconfig getifaddr en0   # macOS
```

---

## Build for Release

**iOS:**
```bash
flutter build ios --release
```
Then archive and distribute via Xcode or `flutter build ipa`.

**Android:**
```bash
flutter build apk --release
# or for Play Store:
flutter build appbundle --release
```
